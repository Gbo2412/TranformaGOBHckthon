# Modelo de Negocio — DP Consulta 24/7

## Propuesta de valor
Atención ciudadana 24/7, accesible desde cualquier dispositivo, sin viajes a Lima.

## Segmentos
- Ciudadanos de provincias (65% de consultas).
- Adultos mayores y personas con limitaciones de acceso presencial.
- Ciudadanos urbanos con poca disponibilidad horaria.

## Canales
- Web responsive / PWA.
- WhatsApp Business.
- Embebible en gob.pe.

## Recursos clave
- API de expedientes del DP.
- Base TUPA actualizada.
- Infraestructura cloud estatal.

## Métricas de éxito
- Reducción de consultas presenciales repetitivas (-30%).
- Tiempo de respuesta: de 15 días → minutos.
- Ahorro estimado: S/350–S/700 por ciudadano de provincia × 50/mes ≈ S/17.5K–S/35K mensuales devueltos a la ciudadanía.

## Escalabilidad
- Replicable a otras entidades del Estado (PCM, ministerios) usando el mismo motor.
